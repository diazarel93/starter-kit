## Références maîtrisées

### Stratégie de contenu
- **Gary Vaynerchuk** — *Jab, Jab, Jab, Right Hook* : donner de la valeur (jabs) avant de demander (right hook). Ratio 80% valeur / 20% vente
- **Seth Godin** — *This Is Marketing* : trouver le minimum viable audience, être remarquable pour les bonnes personnes. "People like us do things like this"
- **Ann Handley** — *Everybody Writes* : chaque contenu est une conversation, pas une diffusion. Le contenu = service rendu
- **Joe Pulizzi** — *Content Inc* : construire une audience avant le produit, devenir THE référence dans une niche
- **Jay Baer** — *Youtility* : être si utile que les gens veulent payer. Le contenu = utilitaire, pas publicitaire

### Growth & Distribution
- **Andrew Chen** — *The Cold Start Problem* : le bootstrap d'une audience, les réseaux à effet réseau
- **Hiten Shah & Marie Prokopets** — distribution is product
- **Lenny Rachitsky** — croissance organique vs payante, les canaux qui scalent
- **David Perell** — *Write of Passage* : construire une audience via l'écriture, le personal brand comme levier

### Algorithmes & Plateformes (état 2025-2026)
- **Instagram** : Reels (reach max), carousel éducatif (saves = signal fort), Stories (engagement proche)
- **TikTok** : hook en 2s obligatoire, watch time > likes, commentaires = viralité
- **LinkedIn** : texte long-form (>1000 chars), dwell time, partages = boost x5, commentaires > likes
- **YouTube** : CTR miniature × watch time = ranking, Shorts pour top of funnel
- **Twitter/X** : thread > tweet isolé, liens = reach pénalisé (mettre en commentaire)
- **Substack** : discovery via recommandations, taux d'ouverture > follower count

## Architecture de contenu

### Les 3 niveaux de contenu

```
TOFU (Top of Funnel) — Découverte
→ Pour : personnes qui ne vous connaissent pas encore
→ Format : Reels, TikTok, tweets viraux, Shorts YouTube
→ Sujet : tendances, insights chocs, opinions tranchées
→ CTA : aucun (juste valeur)
→ Métrique : reach, impressions, nouveaux followers

MOFU (Middle of Funnel) — Considération
→ Pour : personnes qui vous suivent, qui hésitent
→ Format : carrousels, threads, articles, podcasts, vidéos longues
→ Sujet : how-to, cas pratiques, behind the scenes, comparaisons
→ CTA : s'inscrire à la newsletter, télécharger, rejoindre la liste
→ Métrique : saves, clics, taux de conversion

BOFU (Bottom of Funnel) — Conversion
→ Pour : audience chaude, qui vous fait déjà confiance
→ Format : email, DM, offer post, témoignage, case study
→ Sujet : offre, résultats, preuves, urgence réelle
→ CTA : acheter, réserver, s'inscrire
→ Métrique : ventes, RDV pris, conversions
```

### Les Piliers de contenu (Content Pillars)

Définir 3-5 thèmes récurrents qui structurent toute la communication :

```
Exemple Banlieuwood :
Pilier 1 — CINÉMA : culture, références, histoire, analyses de films
Pilier 2 — COULISSES : making-of des séances, productions élèves
Pilier 3 — PARCOURS : témoignages élèves, transformation, avant/après
Pilier 4 — PÉDAGOGIE : les méthodes, pourquoi ça marche, la doctrine
Pilier 5 — TERRITOIRE : le quartier, la culture urbaine, l'ancrage local
```

## Formats par plateforme (ce qui marche en 2026)

### Instagram

**Reels (reach maximal)**
```
Structure :
- 0-2s : hook visuel + texte accrocheur à l'écran
- 2-15s : développement rapide (1 idée / 3s)
- 15-30s : conclusion + micro-CTA oral (pas de lien)
Ratio : 9:16 vertical
Son : original audio ou trending audio
Sous-titres : OBLIGATOIRE (80% regardent sans son)
```

**Carousel éducatif (saves = signal fort)**
```
Slide 1 : Promesse forte ("5 erreurs qui tuent ton court-métrage")
Slides 2-8 : 1 point par slide, visuel simple + texte court
Slide 9 : Récapitulatif
Slide 10 : CTA (save pour plus tard / partage / lien bio)
Format : 1:1 ou 4:5
```

**Stories (maintenir l'engagement)**
```
Fréquence : 3-7 stories/jour
Types : sondages, questions, coulisses, réactions, compte à rebours
Cohérence : même filtre / template = identité visuelle stories
```

### LinkedIn

**Post texte long-form (le plus puissant)**
```
Ligne 1 : hook fort qui force le "voir plus" (question choc, stat inattendue, opinion tranchée)
Lignes 2-15 : développement avec sauts de ligne fréquents (1 idée par ligne)
Dernières lignes : conclusion + 1 question pour activer les commentaires
Lien : en 1er commentaire (pas dans le post — portée pénalisée)
Hashtags : 3-5 max, pertinents
Fréquence : 3-5x/semaine pour construire, 1-2x pour maintenir
```

**Formats qui sur-performent sur LinkedIn :**
- "Ce que j'ai appris en faisant X" (experience narrative)
- "Tout le monde dit X mais en fait c'est Y" (opinion contrariante)
- "Thread / Voici comment [résoudre un problème précis] en 7 étapes" (valeur actionnable)
- "Résultats de mon expérience après X mois" (social proof + récit)

### TikTok

```
Structure gagnante :
- 0-1s : image ou phrase qui accroche immédiatement
- 1-3s : contexte minimal (qui tu es + pourquoi écouter)
- 3-25s : contenu principal rapide (coupes fréquentes)
- 25-30s : surprise / twist / reveal
- 30s : CTA simple
Sons : trending + originalité = combo gagnant
Texte à l'écran : renforcé, jamais redondant avec la parole
Fréquence idéale : 1-3x/jour pour croître, 3-5x/semaine pour maintenir
```

### Newsletter (Substack / Brevo / Resend)

```
Structure performante :
- Objet : 40 chars max, curiosity gap ou bénéfice précis
- Preview : complète l'objet, pas redondant
- Introduction : 1 paragraphe, 3 lignes max (pourquoi cet email aujourd'hui)
- Corps : 1 idée principale développée avec exemples
- Section secondaire : 2-3 ressources ou actu courtes
- CTA : 1 seul, en rapport avec le contenu
- P.S. : toujours (2e élément le plus lu)
Fréquence : hebdomadaire > mensuel (consistance)
Taux d'ouverture cible : >30% = excellent, >20% = bon, <15% = problème
```

## Stratégies de croissance organique

### Build in Public
```
Principe : partager le voyage, pas seulement les succès
Contenu : chiffres, décisions, échecs, pivots, behind the scenes
Audience cible : fondateurs, entrepreneurs, curieux du domaine
Plateforme : Twitter/X + LinkedIn
Format : update hebdomadaire (revenus, utilisateurs, défis)
Risque : concurrents voient — avantage : confiance et communauté > tout
```

### Content Repurposing (système)
```
1 contenu long (article / podcast / vidéo YouTube)
    ↓ découpe en
5 posts LinkedIn / Twitter
    ↓ découpe en
10 stories Instagram
    ↓ transforme en
3 Reels / TikToks
    ↓ résume en
1 email newsletter
```

### Collaboration & Amplification
```
Guest posts / interviews : audience partagée, légitimité croisée
Newsletters : croiser ses newsletters avec celles de la même niche
Podcasts : invité = reach sans effort de production
Co-création : contenu à 2 = double audience exposée
```

### SEO Content (long terme)
```
Stratégie cluster :
- 1 page pillar (3000+ mots) sur le sujet principal
- 10-20 articles clusters qui pointent vers la pillar
- Chaque article = 1 intent précis (informationnel / commercial / transactionnel)
Outils : Ahrefs, Semrush, ou Google Search Console gratuit
Timeline : 6-12 mois pour voir les résultats
```

## Métriques qui comptent

### Vanity metrics (à ignorer)
- Nombre de followers (sans engagement = inutile)
- Likes (seul)
- Reach brut

### Metrics qui comptent vraiment
```
Engagement rate       : (likes + comments + saves + shares) / reach > 3% = bon
Save rate             : saves / reach > 1% = contenu de valeur
Comment ratio         : commentaires / likes > 0.1 = communauté active
Email list growth     : +X abonnés/semaine
Click-through rate    : % qui cliquent sur le lien bio / CTA
Conversion rate       : % de l'audience qui achète / s'inscrit
Revenue per subscriber: revenu total / nombre d'abonnés newsletter
```
