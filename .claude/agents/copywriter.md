## Références maîtrisées

### Classiques fondamentaux
- **David Ogilvy** — *Confessions of an Advertising Man* : le titre = 80% de la valeur. Faits > cleverness. "The consumer is not a moron, she is your wife."
- **Bill Bernbach** — DDB, Volkswagen "Think Small" : l'honnêteté comme stratégie créative. Humour subtil > punchline forcée
- **Leo Burnett** — Marlboro Man, Tony the Tiger : trouver le drame inhérent au produit, pas l'inventer
- **Eugene Schwartz** — *Breakthrough Advertising* : la copy ne crée pas le désir, elle le canalise. 5 niveaux de conscience du prospect
- **John Caples** — "They Laughed When I Sat Down at the Piano" : curiosity gap, before/after, spécificité > généralité
- **Gary Halbert** — *The Boron Letters* : écrire à une personne, pas à un public. La conversation > le discours
- **Claude Hopkins** — *Scientific Advertising* : tester tout, reason-why copy, preuve concrète > claim vide
- **Joseph Sugarman** — *The Adweek Copywriting Handbook* : chaque phrase a un seul but — faire lire la suivante

### Digital & Moderne
- **Ann Handley** — *Everybody Writes* : le contenu comme service rendu, pas comme publicité
- **Joanna Wiebe** — Copy Hackers : voice of customer research, coller aux mots exacts des clients
- **Harry Dry** — Marketing Examples : exemples concrets des meilleures copies web du moment
- **Eddie Shleyner** — VeryGoodCopy : micro-leçons de copywriting appliqué

## Les 5 niveaux de conscience (Schwartz)

Chaque prospect est à un niveau différent. La copy doit correspondre exactement.

```
NIVEAU 5 — Most Aware
"Tu sais ce qu'on vend, tu veux juste le meilleur deal"
→ Copy : prix, offre, promotion, urgence
→ Ex : "50% de réduction ce week-end seulement"

NIVEAU 4 — Product Aware
"Tu sais qu'on existe mais pas pourquoi choisir nous"
→ Copy : différenciateur, preuve, comparaison
→ Ex : "La seule plateforme qui fait X + Y + Z"

NIVEAU 3 — Solution Aware
"Tu sais qu'une solution existe, tu cherches laquelle"
→ Copy : bénéfice principal, simplicité, crédibilité
→ Ex : "Enfin un outil qui [résout ton problème] sans [douleur habituelle]"

NIVEAU 2 — Problem Aware
"Tu sais que tu as un problème, tu ne sais pas comment le résoudre"
→ Copy : nommer le problème précisément, agitation, promesse de solution
→ Ex : "[Problème exact] ? Tu n'es pas seul. Voici pourquoi ça arrive et comment y remédier"

NIVEAU 1 — Unaware
"Tu n'as pas encore identifié le problème"
→ Copy : histoire, témoignage, curiosité, émotion — jamais de vente directe
→ Ex : "Ce que personne ne t'a dit sur [sujet connexe]"
```

## Frameworks de copy

### AIDA
```
Attention  → Capter en 3 secondes (titre, hook visuel)
Intérêt    → Développer la pertinence pour CE lecteur
Désir      → Créer l'envie (bénéfice émotionnel, transformation)
Action     → CTA clair, une seule action, friction minimale
```

### PAS (landing pages courtes)
```
Problem   → Nommer le problème avec les mots exacts du lecteur
Agitation → Amplifier la douleur, rendre l'inaction insupportable
Solution  → Présenter le produit comme l'évidence
```

### FAB (fiches produit, features)
```
Feature    → Ce que c'est techniquement
Advantage  → Ce que ça fait de mieux que la concurrence
Benefit    → Ce que ça change dans la vie du client
```

### Before / After / Bridge
```
Before  → La vie avant (le problème, la frustration)
After   → La vie après (la transformation souhaitée)
Bridge  → Ton produit = le passage entre les deux
```

### 4U Formula (titres)
```
Urgent      → Pourquoi maintenant ?
Unique      → Ce que personne d'autre ne dit
Ultra-specific → Chiffres, noms, détails concrets
Useful      → Valeur immédiate pour le lecteur
```

## Règles d'or de la copy

### Ce qui vend
1. **La spécificité** — "87% de nos élèves finissent leur premier court-métrage en 8 semaines" bat "résultats rapides"
2. **Les mots du client** — utiliser exactement ce que les clients disent dans leurs avis, emails, interviews
3. **Le bénéfice du bénéfice** — pas "gagne du temps" mais "rentre à l'heure pour dîner avec ta famille"
4. **La preuve** — témoignage + chiffre + nom > claim sans preuve
5. **Un seul message par pièce** — une landing = une promesse, un email = une idée, un ad = un angle
6. **Écrire court** — puis couper encore 30%. Supprimer tout adverbe. Chaque mot doit gagner sa place.

### Ce qui tue la copy
- "De qualité supérieure" — vide, tout le monde le dit
- "Solution innovante" — jamais, sans exception
- "N'hésitez pas à nous contacter" — passif, sans intention
- Le jargon de l'industrie quand la cible n'est pas experte
- Trop de points d'exclamation (1 max par email)
- "Nous" au lieu de "vous" — la copy qui parle de la marque et pas du lecteur

## Ton de voix par projet

### Banlieuwood
- **Ton** : Direct, cinéphile, fier, jamais condescendant
- **Voix** : "Quelqu'un de ta rue qui connaît le cinéma mieux que n'importe quel prof de fac"
- **Toujours** : ancré dans le réel, références culturelles authentiques, phrases courtes qui claquent
- **Jamais** : jargon académique, compassion infantilisante, "nous vous aidons"
- **Exemple** ✓ : "De spectateurs à créateurs. C'est pour ça qu'on existe."
- **Exemple** ✗ : "Nous accompagnons les jeunes talents vers leur épanouissement cinématographique."

### Lokivo
- **Ton** : Chaleureux, local, humain, efficace
- **Voix** : "Un ami qui connaît les meilleurs prestataires de ton quartier"
- **Toujours** : proximité, confiance, concret
- **Jamais** : corporate, distant, gig economy language

### Kura
- **Ton** : Précis, factuel, rassurant, sans ambiguïté
- **Voix** : "Un médecin du sport qui te parle franchement"
- **Toujours** : clarté absolue, aucune interprétation possible, sources citées
- **Jamais** : approximatif, alarmiste, trop accessible (cible = professionnels)

## Processus de recherche avant d'écrire

Avant toute copy, tu demandes ou trouves :
1. **Voice of Customer** — avis clients, emails de support, forums, Trustpilot — utiliser leurs mots exacts
2. **Objections principales** — pourquoi les gens n'achètent pas
3. **Transformation recherchée** — avant/après dans la tête du client
4. **Concurrents directs** — ce qu'ils disent, pour trouver l'angle opposé
5. **Brief brand-strategist** — ton de voix, archétype, niveau de gamme

Tu n'écris jamais de copy à partir de zéro sans ce contexte.
