## Comment lire cette cartographie

Chaque style est un **système cohérent** — pas juste une couleur ou une font.
Un vrai style a : palette + typographie + layout + densité + mouvement + émotion.

## 01 — Cinema Dark Premium

**Émotion :** Immersion, mystère, culture, prestige
**Mouvement :** Suisse → A24 → MUBI 2020s

### Codes visuels
- Fond : noir chaud (#0A0905 à #1A1410) — jamais froid
- Texte : off-white (#F5F0EB) — jamais blanc pur
- Accent : or (#D4A843) ou orange brûlé (#FF6B35)
- Typographie : condensed sans-serif (Bebas Neue, Druk, Akzidenz) pour titres + serif élégant pour body
- Layout : asymétrique, image dominante, texte ancré bas-gauche
- Densité : aéré, beaucoup d'espace
- Motion : lent, cinématique, fade > slide

### Créateurs de référence
- A24 Films (a24films.com)
- MUBI (mubi.com)
- Criterion Collection (criterion.com)
- Letterboxd
- Sundance Institute (sundance.org)
- The Criterion Channel

### Studios maîtres de ce style
- **Gretel** — identités Netflix, HBO
- **Imaginary Forces** — générique de films
- **Trollbäck+Company** — branding entertainment

### Ce qui le rend exceptionnel
Le fond n'est JAMAIS `#000000` — le noir froid tue l'émotion. Le noir chaud crée de la profondeur. L'espace blanc (ici : espace noir) est utilisé comme le cinéma utilise le silence.

### À utiliser pour
Festivals, cinéma, musique, luxe, culture, portfolios créatifs

### À éviter pour
Dashboard, EdTech enfants, SaaS B2B, e-commerce

## 02 — SaaS Minimal Premium

**Émotion :** Clarté, confiance, efficacité, modernité
**Mouvement :** Swiss Style → Material → Linear 2022+

### Codes visuels
- Fond : blanc (#FFFFFF) ou gris très clair (#FAFAFA) ou dark (#09090B)
- Texte : quasi-noir (#111827) très haut contraste
- Accent : 1 couleur unique très saturée (violet, bleu, vert) — utilisée avec parcimonie
- Typographie : sans-serif géométrique (Inter, Geist, Söhne) — régulier, précis
- Layout : grille stricte, symétrique, 8px system
- Densité : standard à dense — beaucoup d'info mais organisée
- Motion : rapide, fonctionnel, 150-200ms, ease-out

### Créateurs de référence
- Linear (linear.app)
- Vercel (vercel.com)
- Stripe (stripe.com)
- Raycast (raycast.com)
- Cal.com
- Resend (resend.com)
- Clerk (clerk.dev)

### Studios maîtres de ce style
- **Fantasy** — NY, Stripe, Figma
- **Rauno Freiberg** (indépendant, Vercel)
- **Paco Coursey** (indépendant)
- **Emil Kowalski** (animations React)

### Ce qui le rend exceptionnel
La densité sans chaos. Linear montre qu'on peut mettre beaucoup d'information sans que l'œil soit perdu — tout repose sur la hiérarchie typographique et la grille 8px.

### À utiliser pour
SaaS, outils productivité, dashboard pro, API tools, dev tools

### À éviter pour
Landing émotionnelle, festival, luxe, portfolio créatif

## 03 — Editorial Bold (Type-Forward)

**Émotion :** Impact, opinion, culture, intelligence
**Mouvement :** Neville Brody / The Face → David Carson / Ray Gun → 2020s editorial web

### Codes visuels
- Typographie = élément hero principal (100px+, parfois plein écran)
- Layout : asymétrique, tension entre éléments, lignes de force diagonales
- Couleurs : souvent limitées à 2-3 (noir + 1 accent vif)
- Images : utilisées sparingly, souvent en overlap avec le texte
- Blanc : utilisé de façon dramatique — soit tout vide soit tout plein
- Motion : texte animé, split, scroll-driven

### Créateurs de référence
- Pentagram (pentagram.com)
- Collins (wearecollins.com)
- Build (wearebuild.com)
- It's Nice That (itsnicethat.com)
- Eye Magazine (eyemagazine.com)
- Pitchfork (pitchfork.com)

### Studios maîtres de ce style
- **Bibliotheque** — Londres
- **North** — Londres
- **Studio Dumbar** — Amsterdam
- **Order** — NYC

### Ce qui le rend exceptionnel
Le texte N'EST PAS là pour être lu en premier — il est là pour créer une impression visuelle. On scanne avant de lire. La typographie devient architecture.

### À utiliser pour
Magazine, media, portfolio studio créatif, festival culturel, marque avec fort point de vue

### À éviter pour
App mobile grand public, e-commerce, onboarding utilisateur

## 04 — Luxury White

**Émotion :** Exclusivité, raffinement, intemporalité, désirabilité
**Mouvement :** Chanel → Dior → Apple → luxe digital 2020s

### Codes visuels
- Fond : blanc pur ou crème (#FAF8F5)
- Texte : noir intense ou brun très foncé
- Accent : or, beige, champagne — très discret
- Typographie : serif élégant (Canela, Lyon, Garamond, Didot) pour titres — jamais bold agressif
- Layout : beaucoup d'espace blanc — 70% vide minimum
- Images : plein écran, haute qualité, produit ou portrait
- Densité : très faible — chaque élément respire
- Motion : lent, scroll parallaxe, fondu

### Créateurs de référence
- Dior (dior.com)
- Chanel (chanel.com)
- Bottega Veneta
- The Row
- Aesop (aesop.com)
- Byredo (byredo.com)
- Loro Piana

### Studios maîtres de ce style
- **Wolff Olins** — repositionnement luxe
- **Pentagram** — identités intemporelles
- **Base Design** — Bruxelles, luxe européen

### Ce qui le rend exceptionnel
L'espace blanc EST le message. Dire "on peut se permettre de ne rien mettre ici" communique le luxe mieux que n'importe quel élément graphique.

### À utiliser pour
Luxe, beauté, mode, gastronomie, architecture, joaillerie

### À éviter pour
Startup tech, gaming, EdTech, dashboard

## 05 — Playful Color (Dopamine Design)

**Émotion :** Joie, énergie, progression, récompense
**Mouvement :** Push Pin 1960s → MTV 1990s → Duolingo 2020s

### Codes visuels
- Couleurs : multiples, saturées, contrastées — palette 5+ couleurs
- Typographie : rounded, friendly, sans-serif (Nunito, Poppins, Fredoka)
- Layout : cartes, grilles colorées, bento boxes
- Illustrations : custom, personnages, micro-illustrations
- Motion : bouncy, spring animations, célébrations (confetti)
- Densité : modérée, jamais oppressante

### Créateurs de référence
- Duolingo (duolingo.com)
- Spotify Wrapped
- Notion (interface)
- Arc Browser
- Headspace (headspace.com)
- Monzo (monzo.com)

### Studios maîtres de ce style
- **Superflat** — animations joyeuses
- **Ueno** — San Francisco, product design coloré
- **BASIC** — San Diego, brand coloré

### Ce qui le rend exceptionnel
La couleur a une FONCTION — chaque teinte signifie quelque chose. Duolingo n'utilise pas des couleurs vives pour faire "joli" mais pour signaler progression, succès, attention.

### À utiliser pour
EdTech, santé grand public, app mobile, finance grand public (Monzo), gaming

### À éviter pour
B2B enterprise, compliance, luxe, médical sérieux

## 06 — Neo-Brutaliste

**Émotion :** Brut, honnête, anti-corporate, punk digital
**Mouvement :** Punk 1977 → Emigre magazine 1990s → Neo-brutal web 2022+

### Codes visuels
- Bordures : noires épaisses (2-4px), toujours visibles
- Fond : blanc pur ou couleur très vive (jaune #FFEE00, rouge, bleu électrique)
- Typographie : grotesk bold, uppercase, serré
- Ombres : offset shadow (box-shadow: 4px 4px 0 black) — pas de blur
- Radius : 0 — tout est carré
- Layout : grille visible, alignement rigide ou délibérément cassé
- Motion : transforms directs, pas de ease subtil

### Créateurs de référence
- Gumroad (gumroad.com) — le pionnier web
- Figma (ancienne landing)
- Loom
- Beehiiv (beehiiv.com)
- Framer templates néo-brutal
- Flux Academy

### Studios maîtres de ce style
- **Brutalist Websites** (brutalistwebsites.com) — archive
- **Heco** — Paris, version premium du brutal

### Ce qui le rend exceptionnel
C'est le seul style qui assume ses "défauts". Les bordures épaisses et les ombres offset disent "je ne me cache pas derrière du polish — ce que tu vois est ce que c'est."

### À utiliser pour
Outils créatifs, portfolio indé, startup avec point de vue fort, newsletter, produit pour designers

### À éviter pour
Finance, santé, entreprise, luxe, tout ce qui demande confiance institutionnelle

## 07 — Glassmorphism Mature

**Émotion :** Modernité, depth, sophistication tech
**Mouvement :** Windows Vista (mal fait) → iOS 7 → Apple Vision Pro 2024 (bien fait)

### Codes visuels
- Surfaces : `backdrop-filter: blur(20px)`, fond semi-transparent (rgba blanc ou noir 10-30%)
- Utilisé PARCIMONIEUSEMENT — max 1-2 niveaux dans une page
- Fond : toujours une image ou dégradé derrière (sinon le blur ne sert à rien)
- Bordures : `border: 1px solid rgba(255,255,255,0.2)` très subtil
- Couleurs : sobres, le glass ne supporte pas les accents trop vifs

### Créateurs de référence
- Apple Vision Pro UI
- Arc Browser
- Apple macOS
- Linear (usage très subtil)

### Ce qui le rend exceptionnel (quand bien fait)
Crée une hiérarchie visuelle sans ombre ni bordure visible. La surface glass "flotte" naturellement.

### Pièges à éviter
- Ne JAMAIS empiler 3+ niveaux de blur (perf catastrophique)
- Ne pas utiliser sur fond uni (le blur ne fait rien)
- Ne pas en abuser — 1 niveau max par écran

### À utiliser pour
Overlay, modal, sidebar sur image, header transparent

## 08 — Bento Grid

**Émotion :** Organisation, richesse d'info, moderne, Apple-like
**Mouvement :** Apple iPhone 15 launch page → GitHub → Linear → partout 2024+

### Codes visuels
- Grille modulaire asymétrique — cellules de tailles différentes (1x1, 2x1, 2x2, 1x2)
- Chaque cellule = 1 idée, 1 stat, 1 feature
- Radius : généreux (16-24px) — bien différent du brutal
- Fond : dark premium ou blanc
- Chaque cellule a son propre mini-design

### Créateurs de référence
- Apple (toutes les pages produit récentes)
- GitHub (landing actuelle)
- Linear
- Vercel

### Ce qui le rend exceptionnel
Permet de montrer BEAUCOUP de features sans scroll. L'asymétrie guide le regard vers les éléments importants (les grandes cellules).

### À utiliser pour
Landing features, page produit, portfolio, "about" page, KPIs

## 09 — AI Generative / New Aesthetic

**Émotion :** Futurisme, créativité, exploration, inconnu
**Mouvement :** Glitch art 2010s → GAN art 2018 → Diffusion 2022 → AI-native design 2025+

### Codes visuels
- Textures organiques impossibles (ce que l'œil humain ne peut pas dessiner)
- Couleurs : iridescentes, holographiques, caméléon
- Typographie : variable fonts animées, déformation
- Images : hyperréalistes ou totalement abstraites — pas de milieu
- Motion : morphing, liquid, particle systems
- Fond : souvent très sombre pour faire "sortir" les éléments

### Créateurs de référence
- Refik Anadol (refikanadol.com)
- Memo Akten (memo.tv)
- Sofia Crespo
- Runway ML
- Adobe Firefly campaigns
- Midjourney community top creators

### Studios maîtres de ce style
- **Active Theory** — WebGL immersif
- **Unit9** — production créative interactive
- **Resn** — web expérientiel

### À utiliser pour
Portfolio artistique, expérience culturelle, campagne de lancement, marque tech avant-gardiste

### À éviter pour
Tout produit qui demande lisibilité et confiance

## 10 — Japonais Minimal

**Émotion :** Sérénité, précision, intemporalité, respect
**Mouvement :** Kenya Hara / Muji → ISSEY MIYAKE → digital japonais contemporain

### Codes visuels
- Espace blanc = élément actif, pas vide
- Typographie : ultra-light weights, beaucoup d'espace entre caractères
- Couleurs : quasi-monochromatique, nuances de gris, 1 accent très discret
- Asymétrie subtile — pas d'asymétrie choc, mais pas de symétrie parfaite non plus
- Images : photographie minimaliste, nature, texture
- Densité : extrêmement basse

### Créateurs de référence
- Kenya Hara / Muji (muji.com)
- Issey Miyake (isseymiyake.com)
- Comme des Garçons
- Marimekko
- Uniqlo (version JP)

### Ce qui le rend exceptionnel
Le "ma" (間) — le concept japonais d'espace négatif comme entité positive. L'espace n'est pas ce qui reste après avoir placé les éléments — c'est un élément à part entière.

## Mélanges de styles qui fonctionnent

Ces combinaisons sont utilisées par les meilleurs studios :

- **Cinema Dark + Editorial Bold** → A24, MUBI — le gold standard du cinéma
- **SaaS Minimal + Bento Grid** → Linear, Vercel — dashboard moderne
- **Luxury White + Japonais Minimal** → Aesop, Byredo — beauté/mode premium
- **Playful Color + Bento Grid** → Notion, Arc — productivité friendly
- **Neo-Brutal + Editorial Bold** → Gumroad, Beehiiv — produit avec opinion

## Mélanges à éviter

- Cinema Dark + Playful Color → contradiction émotionnelle
- Luxury White + Neo-Brutal → paradoxe de marque
- Glassmorphism + Neo-Brutal → conflit formel total
